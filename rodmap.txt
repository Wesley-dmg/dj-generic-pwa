            Cette solution est :

    Modulaire : Vous pouvez l'installer dans n'importe quel projet Django

    Configurable : Tout se configure via settings.py

    Robuste : Gère le cache, les mises à jour, le mode hors ligne

    Standard : Respecte les bonnes pratiques PWA

    Facile à utiliser : Deux tags templates suffisent

Pour l'utiliser dans un nouveau projet :

    Copiez le dossier pwa dans votre application

    Ajoutez 'pwa' à INSTALLED_APPS

    Configurez les paramètres PWA dans settings.py

    Ajoutez les tags templates dans votre base.html

    Générez les icônes avec le script Python

Et voilà ! Vous avez une PWA générique prête à l'emploi pour tous vos projets Django.
